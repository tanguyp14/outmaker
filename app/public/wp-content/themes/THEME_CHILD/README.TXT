INSTALATION --------------------------


Étape 1 
 - Instaler le theme au même niveau que le theme parent.


Étape 2
 - Changer dans le style.css la ligne 'Template' avec le nom du dossier du THEME PARENT.


Étape 3
 - Sur le BO de WP, activer le Theme-enfant.


Étape 4
 - Inslaller le Plug-in 'WP SCSS' : 
		   
- Configuration suivantes : 

	
		   
SCSS Location        ->  /scss/
			   
CSS Location         ->  /css/
			   
Compiling mode       ->  Compressed
			   
Error Display        ->  Show in Header
			   
Enqueue Styessheets  ->  NOT ACTIVATED

		
   
- Enregistrer les modifications



Étape 5
 - Vous pouvez à present modifier le fichier /scss/style_perso.scss pour votre CSS sur votre Site World Press
